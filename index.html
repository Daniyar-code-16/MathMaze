01 — Что такое MathMaze?

Инновационный подход к изучению математики через игровую механику

🤔

Проблема

85% школьников считают математику скучной и оторванной от реальности. Традиционные методы обучения теряют эффективность в цифровую эпоху.

💡

Решение

MathMaze превращает решение примеров в захватывающее приключение. Чтобы открыть дверь — реши уравнение. Чтобы победить врага — выбери правильный ответ.

✨

Уникальность

Адаптивная сложность: игра подстраивается под уровень игрока. Чем лучше решаешь — тем сложнее примеры, но и больше награда.

🎯 Для кого эта игра?

👦 Школьники 7-12 лет

Освоение математики в игровой форме без стресса

👨‍🎓 Подростки 13-16 лет

Закрепление знаний и подготовка к экзаменам

👨‍👩‍👧 Семьи

Совместный досуг с пользой для всех

02 — Функциональные требования

Что наша игра должна уметь делать

🎮 Геймплей

  • Управление персонажем в 2D-лабиринте
  • Система дверей, открываемых решением примеров
  • Враги, которых можно победить только правильным ответом
  • Собираемые бонусы и улучшения
  • Система чекпоинтов и сохранений
  • Три уровня сложности (легкий, средний, сложный)

📚 Образовательная часть

  • Адаптивная генерация примеров под возраст
  • Темы: сложение, вычитание, умножение, деление, дроби
  • Статистика успеваемости для родителей
  • Режим тренировки без таймера
  • Подсказки и объяснения ошибок
  • Система достижений за прогресс

👤 Пользовательская часть

  • Регистрация и профили игроков
  • Сохранение прогресса в облаке
  • Таблица лидеров среди друзей
  • Настройки управления и графики
  • Мультиязычность (рус/каз/англ)
  • Родительский контроль времени

03 — Технологический стек

Инструменты, которые мы используем

Python 3.11 Pygame CE Tiled Map Editor SQLite Flask (API) React (dashboard) Docker GitHub Actions Aseprite (пиксель-арт) Figma

🎨 Клиентская часть

Движок: Pygame CE для отличной производительности

Графика: Пиксель-арт в стиле ретро (16-bit)

Звук: Procedural audio через PySynth

☁️ Серверная часть

API: Flask + REST

БД: SQLite (локально) / PostgreSQL (облако)

Аутентификация: JWT-токены

🚀 Деплой

Игра: Itch.io + GitHub Releases

Сайт: GitHub Pages

API: Railway / Render
